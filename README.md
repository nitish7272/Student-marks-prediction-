# 🎓 Student Marks Prediction System

## 📌 Project Overview

The Student Marks Prediction System is a Machine Learning project that predicts a student's final exam marks based on various factors such as study hours, attendance, assignment scores, previous exam marks, and sleep hours.

This project uses **Linear Regression** to train a predictive model and provides a **Tkinter GUI interface** for user interaction.

---

## 🚀 Features

* Predict student marks using Machine Learning
* Easy-to-use GUI interface
* Excel dataset for training
* Fast predictions
* Simple and beginner-friendly ML project

## ⚙️ Installation

1. Clone the repository

git clone https://github.com/yourusername/student-marks-prediction.git

2. Move into project folder

cd student-marks-prediction

3. Install dependencies

pip install -r requirements.txt

---

## ▶️ Run the Project

Step 1: Train the model

python train_model.py

Step 2: Run the GUI

python app.py
